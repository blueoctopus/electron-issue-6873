How to reproduce Electron issue #6873 (happens on OS X using a trackpad):

  1. Click View -> Toggle Developer Tools and watch the Console output
  2. Using the trackpad, perform a scroll gesture on the webview or scroll-touch area
  3. My tests on Electron v1.3.4 show that the events are reported only half of the time on these areas, while Electron v1.3.2 works as expected
  4. On the BrowserWindow area everything works as expected on any version
